# Renderify documentation

Renderify is a runtime-first renderer. It can be used without its built-in LLM
providers: provide a RuntimePlan or reviewed source module and render it in the
browser.

## Start here

- Open the renderer-only Playground to edit code and see it immediately.
- Read the browser integration guide before embedding Renderify in an app.
- Keep untrusted input on the declarative RuntimePlan path.
